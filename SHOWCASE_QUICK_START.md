## 🎯 3 個必做任務（Tier 1 = 65 分）

### 1️⃣ 加入 Cover Image（2 分鐘）
- 打開頁面最上方 → `Add cover`
- 推薦：Unsplash 搜尋 "tech dashboard"（深色科技風）
- 或：Purple-Blue gradient（符合品牌）

### 2️⃣ 嵌入 12 個 Database Views（15-20 分鐘）
- 找到「💎 您將獲得什麼？」section 的 12 個編號列表
- 在每個項目下方：按 Enter → 輸入 `/linked` → 選 database
- **關鍵對應**：
  ```
  1. 產品資料庫 → Products (Gallery view)
  2. Blog 文章庫 → Blog Posts (Table view)
  3. Tech Stack → Tech Stack Registry (Table)
  4. API 端點 → API Endpoints Registry (Table)
  5. Component → Component Library (Table)
  6. Brand → Brand Guidelines (Gallery)
  7. ADR → Architecture Decisions (Board)
  8. Email → Email Templates (Table)
  9. CRM → Ultra Lab - Inquiries (Table)
  10. Probe → UltraProbe - Security Leads (Table)
  11. Topics → Mind Threads - Topics (Board)
  12. Posts → Mind Threads - Posts (Table)
  ```

### 3️⃣ 上傳 2 張截圖（5 分鐘）
- **截圖 1**：內部戰情室整體畫面（顯示 3-4 個 databases）
  - 位置：Hero section 下方
  - 方法：Win + Shift + S → 回到 showcase → `/image` → Upload

- **截圖 2**（可選）：Automation script 執行畫面
  - 位置：「⚡ 自動化」section 下方
  - 內容：Terminal 顯示 "Successfully imported XX items"
